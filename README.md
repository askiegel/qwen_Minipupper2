# Qwen Mini Pupper 2

Professional autonomous robotics software stack for the Mini Pupper 2.

## Project Goals

Build a commercial-style autonomous quadruped capable of:

- Person Following
- Object Following
- Person Re-Identification (ReID)
- SLAM Navigation
- Voice Commands
- Multi-Robot Cooperation

The software is designed around modular autonomy rather than a single monolithic node.

---

## Current Features

- Mission Manager
- Behavior Manager
- Target Manager
- Object Tracker
- PID Follow Controller
- Search Behavior
- LiDAR Safety
- Dashboard Pro
- YOLO Vision Server
- Live Bounding Boxes
- Mission Switching
- Person Tracking
- Backpack Tracking
